## 📸 Ajouter vos captures d'écran

Les guides incluent des espaces réservés pour vos captures:

```

```

**Pour ajouter une image:**

1. Mettez votre screenshot dans le dossier `captures/`
2. Nommez-le clairement: `screenshot_page-principale.png`
3. Insérez le lien Markdown:
   ```markdown

   ```
