## ✅ Opérations courantes

### Créer une nouvelle infrastructure

```
1. Cliquez [➕ NOUVEAU]
2. Remplissez le formulaire:
   ├─ Informations collecte (date, enquêteur, téléphone)
   ├─ Localisation (commune, arrondissement, village, GPS)
   ├─ Caractéristiques (secteur, type, année, matériaux)
   ├─ État (fonctionnement, dégradation, réhabilitation)
   ├─ Observations (défectuosités, mesures proposées)
   └─ Photos (4 fichiers max)
3. Cliquez [💾 ENREGISTRER]
```

### Consulter les détails

```
1. Trouvez l'infrastructure dans le tableau
2. Cliquez [👁️ VOIR]
3. Consultez toutes les informations
4. Galerie de photos avec zoom
```

### Modifier une infrastructure

```
1. Trouvez l'infrastructure
2. Cliquez [✏️ MODIFIER]
3. Changez les champs voulus
4. Cliquez [💾 ENREGISTRER]

⚠️ Note: Vous ne pouvez modifier que vos propres données (sauf si Admin)
```

### Supprimer une infrastructure

```
1. Trouvez l'infrastructure
2. Cliquez [🗑️ SUPPRIMER]
3. ⚠️ Confirmation requise
4. Cliquez [SUPPRIMER]
```

## 💾 Exporter les données

```
SECTION "EXPORTER LES DONNÉES":

1. Sélectionnez les filtres (optionnel)
2. Cliquez le format souhaité:
   ├─ [📊 EXCEL] pour fichier .xlsx
   └─ [📄 PDF] pour document PDF

LE FICHIER EST TÉLÉCHARGÉ AUTOMATIQUEMENT
```

## 📥 Importer des données

```
SI VOUS ÊTES AUTORISÉ:

1. Cliquez [📥 IMPORTER]
2. Modal s'ouvre:
   ├─ Cliquez "Sélectionner un fichier"
   ├─ Choisissez le fichier: .xlsx, .xls, ou .csv
   └─ Cliquez [IMPORTER]

FORMATS ACCEPTÉS:
├─ Excel (.xlsx, .xls)
└─ CSV

APRÈS L'IMPORT:
├─ ✅ X infrastructures importées
├─ ⚠️ Y avertissements
└─ ❌ Z erreurs
```

## ❓ PROBLÈMES COURANTS

### Code d'accès incorrect
```
✓ Vérifiez la casse (MAJUSCULES/minuscules)
✓ Supprimez les espaces inutiles
✓ Consultez l'administrateur
```

### Impossible de modifier une infrastructure
```
✓ Vérifiez que c'est VOTRE infrastructure
✓ Vérifiez votre rôle utilisateur
✓ Contactez un administrateur
```

### Photos ne s'affichent pas
```
✓ Vérifiez le format (JPG, PNG, GIF au moins)
✓ Vérifiez la taille (max 10 MB)
✓ Rafraîchissez la page
```

### Erreur lors de l'importation
```
✓ Vérifiez le format du fichier
✓ Vérifiez la structure des données
✓ Consultez un administrateur
```
