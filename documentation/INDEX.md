## 🎨 AJOUTER VOS CAPTURES D'ÉCRAN

### Localisation des espaces:

Chaque fois que vous voyez:
```
> **📸 ESPACE RÉSERVÉ POUR CAPTURE D'ÉCRAN:**
> 
> Insérez ici une capture d'écran de...
```

Vous pouvez:

1. **Faire la capture** (Print Screen ou screenshot tool)
2. **Nommer le fichier** de manière claire
3. **Insérer l'image** au format Markdown:
   ```
   ![Description image](./captures/screenshot_name.png)
   ```

### Conseils pour les captures:
```
✅ BON:
├─ Capture la zone pertinente seulement
├─ Masquer les informations sensibles
├─ Utiliser une résolution cohérente (1920x1080)
└─ Format PNG ou JPG

❌ MAUVAIS:
├─ Capture d'écran entière (trop grand)
├─ Information personnelle visible
└─ Mauvaise qualité/floue
```
